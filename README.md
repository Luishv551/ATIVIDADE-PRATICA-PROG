# Portfólio — Luis Santos

Portfólio pessoal desenvolvido com **HTML5**, **CSS3** e **JavaScript** puro (sem frameworks), como atividade prática da disciplina **Fundamentos da Programação Web** na FIAP.

🌐 **Site publicado:** https://luishv551.github.io/ATIVIDADE-PRATICA-PROG/

---

## Funcionalidades

- Single Page com navegação por âncoras
- Tema claro / escuro com persistência via `localStorage`
- Menu hamburger responsivo para mobile
- Scroll spy (link ativo conforme seção visível)
- Formulário de contato com validação JavaScript e modal de confirmação
